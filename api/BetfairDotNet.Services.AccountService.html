GetAccountStatement(string?, int?, int?, TimeRange?, IncludeItemEnum?, WalletEnum?)

Fetches an account statement with details such as P/L, deposits, withdrawals, and more.

public Task<AccountStatementReport> GetAccountStatement(string? locale = null, int? fromRecord = null, int? recordCount = null, TimeRange? itemDateRange = null, IncludeItemEnum? includeItem = null, WalletEnum? wallet = null)

Parameters

locale string

Language used; defaults to account's setting.

fromRecord int?

Start index for records; default is 0.

recordCount int?

Max records to return; max limit is 100.

itemDateRange TimeRange

Date range for items; max range is last 90 days.

includeItem IncludeItemEnum?

Items to include; default is ALL.

wallet WalletEnum?

Wallet for statement; default is UK.

Returns

Task<AccountStatementReport>

A task with the account statement details.

Examples

Usage:

var statement = await client.Account.GetAccountStatement(
    locale: "en", 
    recordCount: 10
    itemDateRange: new TimeRange(From = new DateTime(...), To = new DateTime(...))
);
